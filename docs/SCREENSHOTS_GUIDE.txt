Запускай ці команди в терміналі та роби скріншоти вікон:

--------------------------------------------------------------------------------
СКРІНШОТ 1: Встановлення лінтера
--------------------------------------------------------------------------------
Команда:
    pip install ruff mypy

Скріншот: вікно терміналу з успішним встановленням пакетів

--------------------------------------------------------------------------------
СКРІНШОТ 2: Результат ДО виправлення (було 68 помилок)
--------------------------------------------------------------------------------
Команда:
    ruff check . --statistics

Скріншот: вікно терміналу з 68 помилками:
    51	F401	unused-import
    9	F811	redefined-while-unused
    6	F841	unused-variable
    1	E722	bare-except
    1	F541	f-string-missing-placeholders

--------------------------------------------------------------------------------
СКРІНШОТ 3: Результат ПІСЛЯ виправлення (0 помилок)
--------------------------------------------------------------------------------
Команда:
    ruff check .

АБО
    ruff check . --statistics

Скріншот: вікно терміналу з повідомленням "All checks passed!" або порожнім виводом

--------------------------------------------------------------------------------
СКРІНШОТ 4: Конфігурація pyproject.toml
--------------------------------------------------------------------------------
Команда:
    notepad pyproject.toml

Скриншот: вміст файлу конфігурації Ruff

--------------------------------------------------------------------------------
СКРІНШОТ 5: Документація docs/linting.md
--------------------------------------------------------------------------------
Команда:
    notepad docs/linting.md

Скріншот: початок документації

--------------------------------------------------------------------------------
СКРІНШОТ 6: Pre-commit hooks (опціонально)
--------------------------------------------------------------------------------
Команда:
    notepad .pre-commit-config.yaml

Скріншот: вміст файлу .pre-commit-config.yaml

--------------------------------------------------------------------------------
СКРІНШОТ 7: Makefile (опціонально)
--------------------------------------------------------------------------------
Команда:
    notepad Makefile

Скріншот: список команд Makefile

================================================================================
ПОСЛІДОВНІСТЬ ДЛЯ ЗВІТУ:
================================================================================

1. Встановити ruff та mypy (скріншот 1)
2. Запустити ruff check (скріншот 2 - 68 помилок)
3. Виправити помилки
4. Запустити ruff check (скріншот 3 - 0 помилок)
5. Показати конфігурацію (скріншоти 4-7)

Мінімум для здачі:
- Скріншот встановлення
- Скріншот "До" (68 помилок)
- Скріншот "Після" (All checks passed!)
- Конфігурація pyproject.toml
- Документація docs/linting.md
